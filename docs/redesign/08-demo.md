# Демо на лендинге — карта продукта и журнал работ

> Last reviewed: 2026-07-14

Публичное демо на лендинге: пользователь прикрепляет файл (КП / счёт / ТТН) → получает ответ
«что за подрядчик, какие товары и т.п.». Плюс готовые примеры (рус/бел/каз) и возможность
загрузить свой файл. **Дизайн/тексты/UX правим потом — сейчас главное, чтобы всё отрабатывало.**

Этот документ — источник правды по демо. **Перечитывать перед каждым новым куском работы** и
держать журнал внизу синхронно.

## Требования (из задачи владельца)

1. Прикрепить файл → ответ: подрядчик (поставщик), товары, тип документа, суммы.
2. Дать 2–3 примера (или сгенерировать самому): КП, счёт, ТТН на рус/бел/каз.
3. Пользователь может загрузить свой файл — **предупредить о публичности** и **ограничить
   3 файла за 10 минут**.
4. Баннер-текст: «если нужно доработать — сделаем на ваших серверах».
5. Всё должно реально отрабатывать (UX/дизайн — потом).

## Окружение (проверено вживую)

- **Прод-образ несёт AI-конвейер**: бинарники извлечения (`pdftotext`/`libreoffice`/`tesseract`
  rus+bel+kaz+eng) + агент (Claude Code CLI против DeepSeek через Anthropic-совместимый шлюз).
  В dev-контейнере/CI их нет — поэтому AI-путь **проверяется в проде** (как основной конвейер);
  чистое ядро маппинга покрыто юнит-тестами с моками.
- Bitrix24 в демо **не** участвует (ни токенов, ни записи) — только разбор и показ.

## Архитектурное решение (P5-b, #45)

Демо **маршрутизирует по формату** — детерминированно там, где можно, AI там, где нужно:

- **Текст / CSV / XLSX → детерминированный экстрактор** `app/utils/demoExtract.ts` (мгновенно,
  бесплатно, покрыт тестами): тип документа (КП/счёт/ТТН по ключевым словам рус/бел/каз),
  поставщик (метка + налоговый ID УНП/ИНН/БИН/БСН), таблица товаров, итоги, **валюта** (₽ RUB, ₸ KZT, знак НБ РБ для BYN — см. «Значки валют»),
  НДС и «Всего».
- **PDF / скан (.png/.jpg) / Word → реальный AI-конвейер** `server/utils/demoAi.ts`
  (`runDemoAiExtract`): temp-файл → `extractText` (poppler/libreoffice/OCR) → `runAgent` (DeepSeek) →
  чистый маппер `extractedToDemoResult` (те же поля, включая валюту и НДС) → структура **или
  честная ошибка** (никогда 500 на плохой документ). Temp-файл удаляется в `finally`.
- **Backend** `POST /api/demo/extract`: rate-limit 3 файла / 10 мин на IP + **глобальный лимит
  одновременных AI-разборов** (`DEMO_AI_MAX_CONCURRENCY`, деф. 2 → 503 при перегрузке); потолок
  **5 МБ**; флаг публичности; ответ — JSON `DemoResult`. nginx `client_max_body_size 6m` на этом роуте.
- **UI** `app/components/DemoTryout.vue` (секция `#demo`, визуально выделена): дропзона + кнопки
  примеров + **прогресс-бар** разбора + карточка результата (валюта в шапках/итогах).
- **Готовые примеры** — `public/demo/*` (**18 файлов**: КП/счёт/ТТН × ru/be/kk в `.txt` **и** `.pdf`).
  Кнопки-примеры парсят `.txt` **на клиенте** (мгновенно, без лимита); ссылка «скачать PDF» отдаёт
  реалистичный PDF, который можно загрузить обратно (путь AI-конвейера).

## Формат демо-файлов

- **Текстовые** (`.txt`, UTF-8) — детерминированно-парсимые: заголовок с типом и номером, блок
  «Поставщик … УНП/ИНН/БИН …», таблица товаров (Наименование | Кол-во | Ед. | Цена | Сумма) + итоги.
  Реквизиты по странам: **РФ** ИНН+₽+НДС 22%, **РБ** УНП+BYN (знак НБ РБ)+НДС 20%, **КЗ** БСН+₸+ҚҚС 12%.
- **PDF** — генерируются из `.txt` (`pnpm demo:pdfs`, Chromium), коммитим как статику; скачиваемы
  с лендинга и проходят через реальный AI-конвейер при загрузке.
- **Принимаемые загрузки**: `.txt/.csv/.tsv`, `.xlsx`, `.pdf/.png/.jpg/.jpeg/.docx/.doc` до 5 МБ.

## Значки валют

Валюта в карточке результата (шапки колонок и итоги) рисуется компонентом
`app/components/CurrencySign.vue` по ISO-4217 коду (`:code`). Правило простое:

- **Есть Unicode-знак** (RUB → ₽, KZT → ₸, USD → $, EUR → €, …) — рендерим его как **текст**
  через `currencySymbol(code)` из `app/utils/demoExtract.ts`. Копипаст и скринридер работают сами.
- **Нет Unicode-кодпоинта** — единственный такой случай сейчас **BYN**. У белорусского рубля нет
  символа в Unicode, поэтому встраиваем **официальный знак НБ РБ** (трассированный контур,
  `fill="currentColor"` — наследует цвет и кегль текста). Рядом — визуально скрытый, но
  **выделяемый** `<span>BYN</span>`: скринридер озвучивает, копипаст даёт реальный текст, а не пусто.
  Источник глифа — [Wikimedia Commons: BYN_symbol.svg](https://commons.wikimedia.org/wiki/File:BYN_symbol.svg),
  помечен **Public domain** (шаблоны `PD-author`/`PD-Layout`/`PD-icon`); плюс это официальный
  государственный денежный знак — использование свободно, атрибуции не требует.

Как добавить новую валюту:

1. Добавь пару `код → символ` в `currencySymbol` (`app/utils/demoExtract.ts`) — если у валюты есть
   Unicode-знак, этого достаточно, отдельная ветка в `CurrencySign` не нужна.
2. Если Unicode-знака **нет** (как у BYN) — добавь `v-else-if="code === '<КОД>'"` ветку в
   `CurrencySign.vue` со встроенным SVG (заливка `currentColor`, `viewBox` по исходнику,
   `aria-hidden`) **плюс** скрытый выделяемый текст-код для a11y/копипаста. Источник глифа
   (файл + лицензию) укажи комментарием прямо в SVG (как у BYN — ссылка на Wikimedia Commons).
3. Заведи для новой валюты **свой** CSS-класс (по образцу `.byn-sign`, но не переиспользуя его):
   `height`≈`0.9em`, `vertical-align` под шрифт — чтобы глиф сидел на базовой линии наравне с
   текстовыми знаками. У каждого глифа своя метрика, общий класс переиспользовать не стоит.

Не использовать `v-html` для знака и не хардкодить символы в шаблонах карточки — только через
`CurrencySign`, чтобы правило рендера жило в одном месте.

## Безопасность демо

- Публичность: предупреждение в UI + в ответе; файлы не сохраняем дольше обработки
  (temp-файл AI-пути удаляется в `finally`).
- Rate-limit 3/10мин на IP (in-memory окно; за прокси клиент — **последний** хоп
  `X-Forwarded-For`, который nginx дописывает и клиент не может подделать) **плюс** глобальный
  лимит одновременных AI-разборов (`DEMO_AI_MAX_CONCURRENCY`, деф. 2 → 503 при перегрузке).
- Потолок 5 МБ (nginx `client_max_body_size 6m` на роуте); `Content-Length` обязателен (411);
  допустимые форматы по расширению; никакого исполнения загруженного.
- Никаких секретов/Б24 в демо-пути.
